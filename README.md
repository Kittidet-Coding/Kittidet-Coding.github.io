# Kittidet Lakthong - Professional Portfolio

A modern, responsive static portfolio website showcasing technical leadership, IT management expertise, and professional experience in the hospitality and technology sectors.

**Live Portfolio:** https://Kittidet-Coding.github.io/

---

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Professional Styling** - Modern blue/indigo color scheme with smooth animations
- **Portfolio Projects** - Showcase of 4 major career positions with descriptions
- **Skills Section** - Visual proficiency bars for 8 core competencies
- **Contact Information** - Easy ways for recruiters to reach out
- **Social Links** - GitHub and LinkedIn profiles
- **Smooth Navigation** - Smooth scrolling between sections

---

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with flexbox and grid
- **Vanilla JavaScript** - Smooth scrolling functionality
- **GitHub Pages** - Free hosting

## How to Use

### View Locally

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Portfolio will display with full functionality

### Deploy to GitHub Pages

1. Create a new GitHub repository named `Kittidet-Coding.github.io`
2. Push these files to the repository
3. Your portfolio will be live at: `https://Kittidet-Coding.github.io/`

---

## Customization

### Change Content

Edit `index.html` to update:
- Your name and title
- Professional summary
- Portfolio projects
- Skills and proficiency levels
- Contact information
- Social media links

### Change Styling

Modify the `<style>` section in `index.html` to customize:
- Colors (update CSS variables in `:root`)
- Fonts
- Spacing
- Animations
- Responsive breakpoints

### Update Headshot

Replace `headshot.jpg` with your own professional photo (same filename).
